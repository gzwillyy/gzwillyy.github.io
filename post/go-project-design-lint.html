GoиҜӯиЁҖ д№Ӣ йЎ№зӣ®и®ҫи®Ў(жұҮжҖ»)

Posted on 2023-02-03 In Golang
  • жң¬ж–Үж‘ҳиҮӘгҖҠд»Һйӣ¶ејҖеҸ‘дјҒдёҡзә§ Go еә”з”ЁгҖӢ

еҰӮдҪ•жһ„е»әеҮәдёҖдёӘе®Ңж•ҙзҡ„ Go йЎ№зӣ®пјҢе№¶дё”е°ҪеҸҜиғҪең°иҜҙжҳҺејҖеҸ‘иҝҮзЁӢдёӯжҲ‘зҡ„жҖқиҖғгҖӮеёҢжңӣиғҪеӨҹйҖҡиҝҮиҝҷдәӣжҖқиҖғи®©дҪ зҹҘйҒ“еҰӮдҪ•ејҖеҸ‘ Go йЎ№зӣ®пјҢе№¶дё”дәҶи§Је…¶д»–ејҖеҸ‘иҖ…зҡ„ејҖеҸ‘жҖқи·ҜгҖӮ

йЎ№зӣ®з ”еҸ‘жөҒзЁӢд»Ӣз»Қ

еңЁеҠЁжүӢж’ёд»Јз Ғд№ӢеүҚпјҢдҪ жңҖеҘҪе…Ҳйёҹзһ°ж•ҙдёӘ Go йЎ№зӣ®ејҖеҸ‘жөҒзЁӢпјҢйҖҡиҝҮеҜ№жөҒзЁӢзҡ„дәҶи§ЈпјҢеҸҜд»ҘеҮҸе°‘дҪ еңЁеҲқж¬ЎиҝӣиЎҢе®Ңж•ҙ Go йЎ№зӣ®ејҖеҸ‘дёӯзҡ„йӮЈз§ҚвҖңдёҚзҹҘжүҖжҺӘвҖқзҡ„ж„ҹи§үпјҢеҸҜд»Ҙи®©дҪ жҺҢжҸЎдёҡз•ҢжңҖдјҳгҖҒд№ҹжҳҜжңҖж ҮеҮҶзҡ„йЎ№зӣ®ејҖеҸ‘жөҒзЁӢпјҢеўһејәдҪ зҡ„иҮӘдҝЎгҖӮ

дёҡз•ҢйҖҡеёёжҠҠз ”еҸ‘жөҒзЁӢеҲҶдёәе…ӯдёӘйҳ¶ж®өпјҢеҲҶеҲ«жҳҜйңҖжұӮйҳ¶ж®өгҖҒи®ҫи®Ўйҳ¶ж®өгҖҒејҖеҸ‘йҳ¶ж®өгҖҒжөӢиҜ•йҳ¶ж®өгҖҒеҸ‘еёғйҳ¶ж®өгҖҒиҝҗиҗҘйҳ¶ж®өгҖӮе…¶дёӯпјҢејҖеҸ‘дәәе‘ҳйңҖиҰҒеҸӮдёҺзҡ„йҳ¶ж®өжңү 4 дёӘпјҡи®ҫи®Ўйҳ¶ж®өгҖҒејҖеҸ‘йҳ¶ж®өгҖҒжөӢиҜ•йҳ¶ж®өе’ҢеҸ‘еёғйҳ¶ж®өгҖӮз ”еҸ‘жөҒзЁӢеҸҠжөҒзЁӢдёӯжҜҸдёӘйҳ¶ж®өзҡ„е…·дҪ“еҶ…е®№пјҢеҰӮдёӢеӣҫжүҖзӨәпјҡ

дҪңдёәдёҖеҗҚејҖеҸ‘дәәе‘ҳпјҢеңЁи®ҫи®Ўйҳ¶ж®өд№ҹдјҡжңүдёҖдәӣи®ҫи®Ўзұ»е·ҘдҪңйңҖиҰҒдҪ еҺ»е®ҢжҲҗпјҢдҫӢеҰӮпјҡи§„иҢғи®ҫи®ЎгҖҒзӣ®еҪ•з»“жһ„и®ҫи®ЎгҖҒе·ҘдҪңжөҒи®ҫи®ЎгҖҒз ”еҸ‘жөҒзЁӢи®ҫи®ЎгҖҒGo д»Јз Ғз»“жһ„и®ҫи®ЎзӯүпјҢе…¶дёӯжҜ”иҫғйҮҚиҰҒзҡ„и®ҫи®ЎжҳҜи§„иҢғи®ҫи®ЎгҖҒзӣ®еҪ•з»“жһ„и®ҫи®ЎгҖҒGo д»Јз Ғз»“жһ„и®ҫи®ЎгҖӮжҺҘдёӢжқҘпјҢжҲ‘дјҡиҜҰз»Ҷд»Ӣз»ҚеҰӮдҪ•жқҘи®ҫи®ЎиҝҷдәӣеҶ…е®№гҖӮ

и§„иҢғи®ҫи®Ў

и§„иҢғи®ҫи®ЎеҶ…е®№д№ҹжҜ”иҫғеӨҡпјҢйҖҡеёёеҢ…еҗ«д»Јз Ғи§„иҢғгҖҒCommit и§„иҢғгҖҒзүҲжң¬и§„иҢғгҖҒжҺҘеҸЈи§„иҢғгҖҒж—Ҙеҝ—и§„иҢғгҖҒй”ҷиҜҜз Ғи§„иҢғзӯүгҖӮ

  • д»Јз Ғи§„иҢғпјҡ Go д»Јз ҒејҖеҸ‘и§„иҢғгҖӮ

  • Commit и§„иҢғпјҡ Commit и§„иҢғгҖӮ

  • зүҲжң¬и§„иҢғпјҡ зүҲжң¬и§„иҢғгҖӮ

  • жҺҘеҸЈи§„иҢғпјҡ жҺҘеҸЈи§„иҢғгҖӮ

  • ж—Ҙеҝ—и§„иҢғпјҡ ж—Ҙеҝ—и§„иҢғгҖӮ

  • й”ҷиҜҜз Ғи§„иҢғпјҡ й”ҷиҜҜз Ғи§„иҢғгҖӮ

дёҠйқўеҸӘеҲ—дёҫдәҶдёҖдәӣеёёз”Ёзҡ„и§„иҢғпјҢеҜ№дәҺдёҖдёӘйЎ№зӣ®жқҘиҜҙпјҢеҰӮжһңжңүйңҖиҰҒеҸҜд»ҘеҲ¶е®ҡжӣҙеӨҡзҡ„и§„иҢғпјҢдҫӢеҰӮпјҡејҖеҸ‘жөҒзЁӢи§„иҢғгҖҒеҸ‘еёғи§„иҢғгҖҒж ЎйӘҢи§„иҢғзӯүгҖӮиҝҷдәӣи§„иҢғдёҚд»…еҸҜд»ҘжҸҗй«ҳд»Јз ҒиҙЁйҮҸпјҢиҝҳеҸҜд»ҘжҸҗй«ҳз ”еҸ‘ж•ҲзҺҮпјҢйҷҚдҪҺеҗҺжңҹзҡ„д»Јз Ғз»ҙжҠӨжҲҗжң¬е’ҢеӯҰд№ жҲҗжң¬гҖӮ

жҸҗзӨәпјҡеңЁе®һйҷ…ејҖеҸ‘дёӯпјҢеӨ§йғЁеҲҶеӣўйҳҹе’ҢйЎ№зӣ®е…¶е®һд№ҹеҸӘз”ЁеҲ°дәҶиҝҷдәӣи§„иҢғгҖӮз”ҡиҮіпјҢжҲ‘жүҖи§ҒиҝҮзҡ„жӣҙеӨҡзҡ„еӣўйҳҹпјҢеҢ…жӢ¬и…ҫи®ҜгҖҒйҳҝйҮҢеҶ…йғЁзҡ„дёҚе°‘еӣўйҳҹйғҪжІЎжңүеҲ¶е®ҡиҝҮејҖеҸ‘и§„иҢғпјҢжҲ–иҖ…иҜҙдёҖдәӣеӣўйҳҹеҸӘжҳҜеңЁйЎ№зӣ®ејҖе§Ӣж—¶жӣҫз»ҸеҲ¶е®ҡиҝҮи§„иҢғпјҢдҪҶйҡҸзқҖйЎ№зӣ®зҡ„иҝӯд»ЈпјҢдјҡйҒ—еҝҳиҝҷдәӣи§„иҢғгҖӮ

и§„иҢғжҳҜејҖеҸ‘дәәе‘ҳеҲ¶е®ҡзҡ„иҪҜзәҰжқҹпјҢйқ ејҖеҸ‘дәәе‘ҳиҮӘи§үзҡ„еҺ»йҒөе®ҲпјҢе№¶дёҚеӨӘеҸҜиЎҢгҖӮеңЁе®һйҷ…ејҖеҸ‘дёӯпјҢжҲ‘д»¬иҰҒеӨҡжҖқиҖғжҖҺд№ҲйҖҡиҝҮе·Ҙе…·еҺ»дҝқиҜҒиҝҷдәӣи§„иҢғиў«йҒөе®ҲпјҢдҫӢеҰӮпјҡеҸҜд»ҘйҖҡиҝҮ golangci-lint иҝӣиЎҢйқҷжҖҒд»Јз ҒжЈҖжҹҘпјҢжқҘдҝқиҜҒд»Јз Ғи§„иҢғпјӣйҖҡиҝҮ gitlint е·Ҙе…·жқҘдҝқиҜҒ Commit и§„иҢғпјӣйҖҡиҝҮ gsemver е·Ҙе…·жқҘз”ҹ SemVer ж јејҸзҡ„зүҲжң¬еҸ·пјӣйҖҡиҝҮ OpenAPI зј–иҫ‘еҷЁжқҘзј–еҶҷ OpenAPI жҺҘеҸЈж–ҮжЎЈпјӣйҖҡиҝҮи°ғз”Ёз»ҹдёҖзҡ„иҜ·жұӮиҝ”еӣһж–№жі•пјҢжқҘзЎ®дҝқй”ҷиҜҜз Ғи§„иҢғпјӣйҖҡиҝҮејҖеҸ‘и§„иҢғеҢ–зҡ„ж—Ҙеҝ—еҢ…пјҢжқҘдҝқиҜҒж—Ҙеҝ—и§„иҢғзӯүгҖӮ

зӣ®еҪ•з»“жһ„и®ҫи®Ў

зӣ®еҪ•з»“жһ„жҳҜдёҖдёӘйЎ№зӣ®зҡ„й—ЁйқўгҖӮеҫҲеӨҡж—¶еҖҷпјҢж №жҚ®зӣ®еҪ•з»“жһ„е°ұиғҪзңӢеҮәејҖеҸ‘иҖ…еҜ№иҝҷй—ЁиҜӯиЁҖзҡ„жҺҢжҸЎзЁӢеәҰгҖӮжүҖд»ҘпјҢеңЁжҲ‘зңӢжқҘпјҢйҒөеҫӘдёҖдёӘеҘҪзҡ„зӣ®еҪ•и§„иҢғпјҢжҠҠд»Јз Ғзӣ®еҪ•и®ҫи®Ўзҡ„еҸҜз»ҙжҠӨгҖҒеҸҜжү©еұ•пјҢз”ҡиҮіжҜ”ж–ҮжЎЈи§„иҢғгҖҒCommit и§„иҢғйғҪиҰҒйҮҚиҰҒгҖӮйҖҡеёёпјҢж №жҚ®еҠҹиғҪпјҢеҸҜд»Ҙе°Ҷзӣ®еҪ•з»“жһ„еҲҶдёәд»ҘдёӢдёӨз§ҚгҖӮ

  • е№ій“әејҸзӣ®еҪ•з»“жһ„пјҡ дё»иҰҒз”ЁеңЁ Go еҢ…дёӯпјҢзӣёеҜ№з®ҖеҚ•гҖӮ

  • з»“жһ„еҢ–зӣ®еҪ•з»“жһ„пјҡ дё»иҰҒз”ЁеңЁ Go еә”з”ЁдёӯпјҢзӣёеҜ№еӨҚжқӮгҖӮ

е№ій“әејҸзӣ®еҪ•з»“жһ„

дёҖдёӘ Go йЎ№зӣ®еҸҜд»ҘжҳҜдёҖдёӘеә”з”ЁпјҢд№ҹеҸҜд»ҘжҳҜдёҖдёӘд»Јз Ғеә“пјҢеҪ“йЎ№зӣ®жҳҜд»Јз Ғеә“ж—¶пјҢжҜ”иҫғйҖӮеҗҲйҮҮз”Ёе№ій“әејҸзӣ®еҪ•з»“жһ„гҖӮ

е№ій“әж–№ејҸе°ұжҳҜеңЁйЎ№зӣ®зҡ„ж №зӣ®еҪ•дёӢеӯҳж”ҫйЎ№зӣ®зҡ„д»Јз ҒпјҢж•ҙдёӘзӣ®еҪ•з»“жһ„зңӢиө·жқҘжӣҙеғҸжҳҜдёҖеұӮзҡ„пјҢиҝҷз§Қж–№ејҸеңЁеҫҲеӨҡеә“дёӯеӯҳеңЁпјҢдҪҝз”Ёиҝҷз§Қж–№ејҸзҡ„еҘҪеӨ„жҳҜеј•з”Ёи·Ҝеҫ„й•ҝеәҰжҳҺжҳҫеҮҸе°‘гҖӮдҫӢеҰӮ glog еҢ…е°ұжҳҜе№ій“әејҸзҡ„пјҢзӣ®еҪ•еҶ…е®№еҰӮдёӢпјҡ

еҜ№дәҺдёҖдәӣеҠҹиғҪз®ҖеҚ•зҡ„е·Ҙе…·жҲ–йЎ№зӣ®пјҢд№ҹеҸҜд»ҘйҮҮз”Ёе№ій“әејҸзҡ„зӣ®еҪ•з»“жһ„пјҢд»ҘеҮҸе°ҸејҖеҸ‘иҖ…зҡ„зҗҶи§ЈжҲҗжң¬е’Ңзј–еҶҷеӨҚжқӮеәҰгҖӮ

з»“жһ„еҢ–зӣ®еҪ•з»“жһ„

еҪ“еүҚ Go зӨҫеҢәжҜ”иҫғжҺЁиҚҗзҡ„з»“жһ„еҢ–зӣ®еҪ•з»“жһ„жҳҜ project-layoutгҖӮиҷҪз„¶е®ғе№¶дёҚжҳҜе®ҳж–№е’ҢзӨҫеҢәзҡ„и§„иҢғпјҢдҪҶеӣ дёәз»„з»Үж–№ејҸжҜ”иҫғеҗҲзҗҶпјҢиў«еҫҲеӨҡ Go ејҖеҸ‘дәәе‘ҳжҺҘеҸ—гҖӮжүҖд»ҘпјҢжҲ‘д»¬еҸҜд»ҘжҠҠе®ғеҪ“дҪңжҳҜдёҖдёӘдәӢе®һдёҠзҡ„и§„иҢғгҖӮжң¬е®һжҲҳйЎ№зӣ® miniblog зӣ®еҪ•д№ҹйҒөеҫӘдәҶ project-layout йЎ№зӣ®зҡ„зӣ®еҪ•и§„иҢғгҖӮ

йЎ№зӣ®д»Јз Ғи®ҫи®ЎйҒөеҫӘз®ҖжҙҒжһ¶жһ„и®ҫи®ЎпјҢдёҖдёӘз®ҖжҙҒжһ¶жһ„е…·жңүд»ҘдёӢ 5 дёӘзү№жҖ§пјҡ

  • зӢ¬з«ӢдәҺжЎҶжһ¶пјҡ иҜҘжһ¶жһ„дёҚдјҡдҫқиө–дәҺжҹҗдәӣеҠҹиғҪејәеӨ§зҡ„иҪҜд»¶еә“еӯҳеңЁгҖӮиҝҷеҸҜд»Ҙи®©дҪ дҪҝз”Ёиҝҷж ·зҡ„жЎҶжһ¶дҪңдёәе·Ҙе…·пјҢиҖҢдёҚжҳҜи®©дҪ зҡ„зі»з»ҹйҷ·е…ҘеҲ°жЎҶжһ¶зҡ„зәҰжқҹдёӯгҖӮ

  • еҸҜжөӢиҜ•жҖ§пјҡ дёҡеҠЎи§„еҲҷеҸҜд»ҘеңЁжІЎжңү UIгҖҒж•°жҚ®еә“гҖҒWeb жңҚеҠЎжҲ–е…¶д»–еӨ–йғЁе…ғзҙ зҡ„жғ…еҶөдёӢиҝӣиЎҢжөӢиҜ•пјҢеңЁе®һйҷ…зҡ„ејҖеҸ‘дёӯпјҢжҲ‘д»¬йҖҡиҝҮ Mock жқҘи§ЈиҖҰиҝҷдәӣдҫқиө–гҖӮ

  • зӢ¬з«ӢдәҺUI пјҡ еңЁж— йңҖж”№еҸҳзі»з»ҹе…¶д»–йғЁеҲҶзҡ„жғ…еҶөдёӢпјҢUI еҸҜд»ҘиҪ»жқҫең°ж”№еҸҳгҖӮдҫӢеҰӮпјҢеңЁжІЎжңүж”№еҸҳдёҡеҠЎи§„еҲҷзҡ„жғ…еҶөдёӢпјҢWeb UI еҸҜд»ҘжӣҝжҚўдёәжҺ§еҲ¶еҸ° UIгҖӮ

  2. жҺ§еҲ¶еұӮпјҲControllerпјүпјҡжҺ§еҲ¶еұӮжҺҘж”¶ HTTP иҜ·жұӮпјҢе№¶иҝӣиЎҢеҸӮж•°и§ЈжһҗгҖҒеҸӮж•°ж ЎйӘҢгҖҒйҖ»иҫ‘еҲҶеҸ‘еӨ„зҗҶгҖҒиҜ·жұӮиҝ”еӣһиҝҷдәӣж“ҚдҪңгҖӮжҺ§еҲ¶еұӮдјҡе°ҶйҖ»иҫ‘еҲҶеҸ‘з»ҷдёҡеҠЎеұӮпјҢдёҡеҠЎеұӮеӨ„зҗҶеҗҺиҝ”еӣһпјҢиҝ”еӣһж•°жҚ®еңЁжҺ§еҲ¶еұӮдёӯиў«ж•ҙеҗҲеҶҚеҠ е·ҘпјҢжңҖз»Ҳиҝ”еӣһз»ҷиҜ·жұӮж–№гҖӮжҺ§еҲ¶еұӮзӣёеҪ“дәҺе®һзҺ°дәҶдёҡеҠЎи·Ҝз”ұзҡ„еҠҹиғҪгҖӮе…·дҪ“жөҒзЁӢеҰӮдёӢеӣҫжүҖзӨәпјҡ
  3. дёҡеҠЎеұӮ (Biz)пјҡдёҡеҠЎеұӮдё»иҰҒз”ЁжқҘе®ҢжҲҗдёҡеҠЎйҖ»иҫ‘еӨ„зҗҶпјҢжҲ‘д»¬еҸҜд»ҘжҠҠжүҖжңүзҡ„дёҡеҠЎйҖ»иҫ‘еӨ„зҗҶд»Јз Ғж”ҫеңЁдёҡеҠЎеұӮгҖӮдёҡеҠЎеұӮдјҡеӨ„зҗҶжқҘиҮӘжҺ§еҲ¶еұӮзҡ„иҜ·жұӮпјҢе№¶ж №жҚ®йңҖиҰҒиҜ·жұӮд»“еә“еұӮе®ҢжҲҗж•°жҚ®зҡ„ CURD ж“ҚдҪңгҖӮдёҡеҠЎеұӮеҠҹиғҪеҰӮдёӢеӣҫжүҖзӨәпјҡ
  4. д»“еә“еұӮпјҲStore)пјҡд»“еә“еұӮз”ЁжқҘи·ҹж•°жҚ®еә“/з¬¬дёүж–№жңҚеҠЎиҝӣиЎҢ CURD дәӨдә’пјҢдҪңдёәеә”з”ЁзЁӢеәҸзҡ„ж•°жҚ®еј•ж“ҺиҝӣиЎҢеә”з”Ёж•°жҚ®зҡ„иҫ“е…Ҙе’Ңиҫ“еҮәгҖӮиҝҷйҮҢйңҖиҰҒжіЁж„ҸпјҢд»“еә“еұӮд»…еҜ№ж•°жҚ®еә“/з¬¬дёүж–№жңҚеҠЎжү§иЎҢ CRUD ж“ҚдҪңпјҢдёҚе°ҒиЈ…д»»дҪ•дёҡеҠЎйҖ»иҫ‘гҖӮиҝҷдёҖеұӮд№ҹдјҡиө·еҲ°ж•°жҚ®иҪ¬жҚўзҡ„дҪңз”Ёпјҡе°Ҷд»Һж•°жҚ®еә“/еҫ®жңҚеҠЎдёӯиҺ·еҸ–зҡ„ж•°жҚ®иҪ¬жҚўдёәжҺ§еҲ¶еұӮгҖҒдёҡеҠЎеұӮиғҪиҜҶеҲ«зҡ„ж•°жҚ®з»“жһ„пјҢе°ҶжҺ§еҲ¶еұӮгҖҒдёҡеҠЎеұӮзҡ„ж•°жҚ®ж јејҸиҪ¬жҚўдёәж•°жҚ®еә“жҲ–еҫ®жңҚеҠЎиғҪиҜҶеҲ«зҡ„ж•°жҚ®ж јејҸгҖӮ


еұӮд№Ӣй—ҙзҡ„йҖҡдҝЎ

дёҠйқўд»Ӣз»ҚдәҶ 4 еұӮз»“жһ„пјҢжҺҘдёӢжқҘжҲ‘д»¬еҶҚзңӢзңӢжҜҸдёҖеұӮд№Ӣй—ҙжҳҜеҰӮдҪ•йҖҡдҝЎзҡ„гҖӮ

йҷӨдәҶжЁЎеһӢеұӮпјҢжҺ§еҲ¶еұӮгҖҒдёҡеҠЎеұӮгҖҒд»“еә“еұӮд№Ӣй—ҙйғҪжҳҜйҖҡиҝҮжҺҘеҸЈиҝӣиЎҢйҖҡдҝЎзҡ„гҖӮйҖҡиҝҮжҺҘеҸЈйҖҡдҝЎпјҢдёҖж–№йқўеҸҜд»ҘдҪҝзӣёеҗҢзҡ„еҠҹиғҪж”ҜжҢҒдёҚеҗҢзҡ„е®һзҺ°пјҲд№ҹе°ұжҳҜиҜҙе…·жңүжҸ’д»¶еҢ–иғҪеҠӣпјүпјҢеҸҰдёҖж–№йқўд№ҹдҪҝеҫ—жҜҸдёҖеұӮзҡ„д»Јз ҒеҸҳеҫ—еҸҜжөӢиҜ•гҖӮ

д»Јз ҒжөӢиҜ•

жҺ§еҲ¶еұӮгҖҒдёҡеҠЎеұӮе’Ңд»“еә“еұӮд№Ӣй—ҙжҳҜйҖҡиҝҮжҺҘеҸЈжқҘйҖҡдҝЎзҡ„гҖӮйҖҡиҝҮжҺҘеҸЈйҖҡдҝЎжңүдёҖдёӘеҘҪеӨ„пјҢе°ұжҳҜеҸҜд»Ҙи®©еҗ„еұӮеҸҳеҫ—еҸҜжөӢгҖӮйӮЈжҺҘдёӢжқҘпјҢжҲ‘д»¬е°ұжқҘзңӢдёӢеҰӮдҪ•жөӢиҜ•еҗ„еұӮзҡ„д»Јз ҒгҖӮиҝҷйҮҢеҸӘд»Ӣз»ҚдёӢжөӢиҜ•жҖқи·ҜгҖӮ

  1. жЁЎеһӢеұӮ: еӣ дёәжЁЎеһӢеұӮдёҚдҫқиө–е…¶д»–д»»дҪ•еұӮпјҢжҲ‘д»¬еҸӘйңҖиҰҒжөӢиҜ•е…¶дёӯе®ҡд№үзҡ„з»“жһ„еҸҠе…¶еҮҪж•°е’Ңж–№жі•еҚіеҸҜгҖӮ
  2. жҺ§еҲ¶еұӮ: жҺ§еҲ¶еұӮдҫқиө–дәҺдёҡеҠЎеұӮпјҢж„Ҹе‘ізқҖиҜҘеұӮйңҖиҰҒдёҡеҠЎеұӮжқҘж”ҜжҢҒжөӢиҜ•гҖӮдҪ еҸҜд»ҘйҖҡиҝҮ golang/mock жқҘ mock дёҡеҠЎеұӮ
  3. дёҡеҠЎеұӮ: еӣ дёәиҜҘеұӮдҫқиө–дәҺд»“еә“еұӮпјҢж„Ҹе‘ізқҖиҜҘеұӮйңҖиҰҒд»“еә“еұӮжқҘж”ҜжҢҒжөӢиҜ•гҖӮдҪ еҸҜд»ҘйҖҡиҝҮ golang/mock жқҘ mock д»“еә“еұӮ
  4. д»“еә“еұӮ: д»“еә“еұӮдҫқиө–дәҺж•°жҚ®еә“пјҢеҰӮжһңи°ғз”ЁдәҶе…¶д»–еҫ®жңҚеҠЎпјҢйӮЈиҝҳдјҡдҫқиө–з¬¬дёүж–№жңҚеҠЎгҖӮжҲ‘д»¬еҸҜд»ҘйҖҡиҝҮ sqlmock жқҘжЁЎжӢҹж•°жҚ®еә“иҝһжҺҘпјҢйҖҡиҝҮ httpmock жқҘжЁЎжӢҹ HTTP иҜ·жұӮгҖӮ

е°Ҹз»“

жҜ”иҫғйҮҚиҰҒзҡ„е·ҘдҪңжңүпјҡи§„иҢғи®ҫи®ЎгҖҒзӣ®еҪ•з»“жһ„и®ҫи®Ўе’Ңд»Јз Ғз»“жһ„и®ҫи®ЎгҖӮе…¶дёӯи§„иҢғзұ»пјҢжң¬иҜҫзЁӢйҮҮз”ЁдәҶдёҡз•Ңзҡ„жңҖдҪіи§„иҢғе®һи·өпјҢзӣ®еҪ•з»“жһ„йҮҮз”ЁдәҶеҪ“еүҚжңҖеҸ— Go ејҖеҸ‘иҖ…ж¬ўиҝҺзҡ„ project-layout зӣ®еҪ•з»“жһ„пјҢд»Јз Ғз»“жһ„йҮҮз”ЁдәҶз®ҖжҙҒжһ¶жһ„гҖӮ
